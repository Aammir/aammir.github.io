NTIBN

The NTIBN or Northern Territory Indigenous Business Network, is website for a group of Indigenous businesses operating in Northern Australia.



   Northern Territory Indigenous Business Network was a Wordpress project.

In this project I applied the following:

  • Made a custom theme using html template.
  • Made a custom login/registration system.
  • Installed and customized a plugin dealing with events.
  • Integrated Stripe payment system to purchase membership, events and query reports.
  • Added custom theme settings.
  • Built a module on admin panel that can manage users.
  • Another module to check the payments
  • Added module for users to download pdf reports after payments.
  • Added module for payments in the admin panel.
  • Customized the admin panel look & feel.
